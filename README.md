# 📐 Pretext-October

Typography and layout discipline for Discord and messaging applications.

## What is this?

A design system adapted from [Cheng Lou's Pretext](https://github.com/chenglou/pretext) — DOM-free text measurement, tight multiline layout, zero reflow.

## Core Principles

| Principle | Meaning |
|-----------|---------|
| **Two-Phase** | Structure once; render clean |
| **Shrink-Wrap** | Fit to content, not arbitrary containers |
| **Predictable Lines** | Consistent line counts |
| **Zero Measurement** | Pre-calculate, don't reflow |

## Patterns

- **Compact Tables** — Align to content width
- **Status Lists** — Visual scan lines
- **Ticker Format** — Fixed-width timestamps
- **Comparison Blocks** — Aligned change markers
